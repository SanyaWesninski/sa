# Калькулятор подсетей

Этот проект представляет собой веб-приложение для расчёта параметров IP-подсетей: адреса сети, количества доступных адресов и других данных на основе введённого IP-адреса и маски подсети.

## Функциональность

- Ввод IP-адреса 
- Выбор маски подсети из списка 
- Валидация введённого IP-адреса (проверка формата и диапазона 0-255 для каждого октета).
- Кнопка "Рассчитать", активная только при валидном IP.
- Отображение результатов:
  - Введённый IP-адрес.
  - Выбранная маска подсети.
  - Рассчитанный адрес сети.
  - Количество возможных адресов в подсети.
- Дизайн: тёмная тема с неоновыми элементами, использующая кастомную UI-библиотеку.
